# SecureBrainBox

> Your private second brain that never forgets.

**100% local AI agent** for Telegram with vector + graph memory.

## Quick Install

```bash
npx securebrainbox install
```

Or install globally:

```bash
npm install -g securebrainbox
sbb install
```

## Requirements

- Docker & Docker Compose
- Node.js 18+
- Python 3.11+ (auto-installed)

## What It Does

1. **Index anything** - Send PDFs, images, audio, URLs, or text via Telegram
2. **Search semantically** - Ask questions in natural language
3. **Connect ideas** - Knowledge graph finds connections you'd miss
4. **100% private** - Everything runs locally, no cloud

## Commands

| Command | Description |
|---------|-------------|
| `sbb install` | Setup wizard |
| `sbb start` | Start services |
| `sbb stop` | Stop services |
| `sbb status` | Check status |
| `sbb logs -f` | View logs |
